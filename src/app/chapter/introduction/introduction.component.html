About Angular

Angular is a JavaScript front-end framework created by Google. Look it up here. There is great documentation including a tutorial. This primer was created using version {{ version }} of Angular.

AngularJS vs Angular 2

Google made a huge change between versions 1 and 2 of the Angular framework. Therefore, the first version is referred to as AngularJS and versions 2 and up as just Angular.

JavaScript and TypeScript

TypeScript is a superset of JavaScript. Browsers do not support TypeScript. So that is why all TypeScript code needs to be compiled down to JavaScript when used in web-based application. Although it is possible to write JavaScript in Angular it is highly recommended to use TypeScript because it is typesafe for one (as the name suggests) and supports object-oriented programming concepts.

Development environment

Use WebStorm (30 day trial / free for teachers and students) from JetBrains or Visual Studio Code, the free IDE from Microsoft for web development. Another option is to use the in-browser IDE StackBlitz, based on VS Code, which is great for prototyping or giving lectures.

Prerequisites

Some knowledge on HTML, CSS, JavaScript and how the DOM works are recommended. Also some hands on experience with OOP is a big plus.

You will need node.js since the Node Package Manager is used to install Angular and required packages (like Bootstrap for instance).
